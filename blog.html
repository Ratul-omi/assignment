What is Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, specifically HTML and XML documents. It represents the page so that programs (such as JavaScript) can interact with and manipulate the structure, style, and content of a webpage.

What is Event delegation in context of the DOM , why is it useful

Event delegation is a technique in JavaScript where you take advantage of event bubbling to handle events at a higher level in the DOM rather than attaching separate event listeners to individual child elements. Instead of adding an event listener to multiple similar elements, you add a single event listener to their parent or ancestor and use logic to handle the events for specific child elements.
